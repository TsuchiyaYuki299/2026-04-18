# てづちゃんとの思い出 - 楽園でまた会う日まで

https://tezu-memories.tsuchiyayuki299.workers.dev/

てづちゃんとの大切な思い出の写真を持ち寄り、みんなで共有するためのプライベートなWebサイトです。

聖書が約束する地上の楽園で、てづちゃんが元気に目ざめるその日まで、私たちのかけがえのない記憶をここに記録し、大切に保管します。

## ✨ サイトの特徴

- **安心のプライベート空間**
  Basic認証（合言葉）を導入し、限られた友人や仲間だけが安心して閲覧・投稿できる設計にしています。
- **楽園をイメージした温かいデザイン**
  パステルカラーを基調とし、ピンクのお花や爽やかな若葉など、美しく穏やかな自然のモチーフで思い出の写真を優しく包み込みます。
- **エピソードを添えて保存**
  写真だけでなく「いつ、どんな場面？」というメッセージを一緒に記録できるため、当時の情景や気持ちを色鮮やかに振り返ることができます。

## 🛠️ 使用技術 (Tech Stack)

このサイトは、高速で安全なエッジコンピューティング環境と、モダンなWeb技術を組み合わせて構築されています。

- **Framework:** [Hono](https://hono.dev/)
- **Infrastructure:** [Cloudflare Workers](https://workers.cloudflare.com/)
- **Database:** Cloudflare D1 (SQLite)
- **Storage:** Cloudflare R2 (Object Storage)
- **Styling:** Tailwind CSS
